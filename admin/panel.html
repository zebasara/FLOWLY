☢️ Reset de Base de Datos

⚠️ zona de peligro — solo para pasar de testing a producción

Este botón resetea a CERO todos los usuarios que no sean admin.
Se borran: tokens, km, niveles, insignias, rachas, misiones, progreso de video y campeón semanal.

Se conserva: nombre, email, ciudad, foto, alias de cobro, referidoPor y estado admin.
Esta acción NO se puede deshacer. Usala solo cuando la app pase a producción real.

Qué se resetea por usuario:
tokensActuales → 0  ·  nivel → 1  ·  kmTotales → 0  ·  move30Dias → 0
badges → []  ·  diasConsecutivosVideos → 0  ·  moveEnHolding → 0
videosCompletadosTotales → 0  ·  videoQuizAnsweredVersion → ""
misionesReclamadasHoy → []  ·  campeonSemanalRacha → 0
También limpia: config/campeon (campeón semanal activo)
